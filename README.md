 # News Aggregator App

<ul> Android application for reading news. </ul>
<ul> Application is using https://newsdata.io website API 
to get news data. </ul>

## App includes: 
- Greetings screen 
- Main screen 
- Single news screen 
- Room database for store local cash data
- Retrofit library for internet requst

## Additional:
- Screens navigation 
- Hilt
