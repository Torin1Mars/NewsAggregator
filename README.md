 # News Aggregator App

<ul> Android application for reading news. </ul>
<ul> Application is using https://newsdata.io website API 
to get news data. </ul>
<ul> Data is storaging localy into local Room Db. </ul>

<tr>
  <td colspan="X">&nbsp;</td> 
</tr>

<ul>Config : </ul>
<ul> Kotlin Version = 2.2.0 </ul>
<ul> AGP Version = 8.8.0-alpha05 </ul>
<ul> CompileSdk = 35 </ul>

## App includes: 
- Greetings screen 
- Main screen 
- Single news screen 
- Room database for store local cash data
- Retrofit library for internet requst

## Additional:
- Screens navigation 
- Hilt
